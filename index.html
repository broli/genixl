genixl by broli

ixl file generator

An application to easily create ixl files to feed into iSiloX
http://www.isilox.com/ iSiloX is the desktop application that converts content to the iSiloâ„¢ 3.x/4.x document format. Unfortunately, the Linux version of iSiloX only works by creating the ixl file by hand (aka text editor This app will make it easy and interactive If you want to read more about the ixl format, head to this site http://www.isilox.com/support/manual/ixl/ixl.htm this is my first semi-serious project. I'm open to ANY suggestions, corrections and help in general, as long as you keep it civilized. carlos.ferrabone {at) gmail [dot} com

Dependencies

These standard GNU C libs stdio.h stdlib.h string.h locale.h iconv.h Mini XML (mxml.h) http://www.minixml.org/

Install

you need to have a compiler, and the standard c libraries also need to have the includes of mini-xml http://www.minixml.org then, standing in the folder, make mkdir make if in doubt, drop me a line at carlos.ferrabone {at) gmail [dot} com or in the github web page https://github.com/broli/genixl/
